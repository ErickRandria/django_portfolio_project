Project 1: IBM DevOps Capstone Project

In this comprehensive Capstone project, you will showcase your DevOps and Software Engineering expertise by developing a secure, microservices-based application deployed on the Cloud. This project integrates the skills and techniques learned throughout the IBM DevOps Engineering Professional Certificate.

Key Components:

  • Agile Planning: Develop a detailed Agile plan by creating user stories and a Kanban board using ZenHub and GitHub.
  • Application Development: Build a RESTful microservice-based application using Flask, applying Test Driven Development (TDD) to ensure robust and reliable code.
  • Continuous Integration and Deployment: Enhance the application with additional functionalities using Continuous Integration, and establish a seamless CI/CD pipeline for automated testing and deployment.
  • Security and Deployment: Implement security best practices, and deploy the application on Cloud using Kubernetes, ensuring scalability and resilience.
  • Monitoring and Improvement: Continuously monitor and enhance the application over multiple sprints.

This project represents the culmination of your learning journey in the IBM DevOps Engineering Professional Certificate, demonstrating your ability to apply advanced DevOps practices in a real-world scenario.

Explore the project on GitHub: GitHub Repository
